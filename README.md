# 841-OTA-Downgrader
 Script that can be used to downgrade almost any 32-bit device to iOS 8.4.1

### Tools used by this script:
- cURL
- partialZipBrowser
- tsschecker
- bsdiff (bspatch)
- kloader
- futurerestore

### Tested on:
- iPad3,3

### Supported devices:
- iPad2,1
- iPad2,2
- iPad2,3
- iPad2,4
- iPad2,5
- iPad2,6
- iPad2,7
- iPad3,1
- iPad3,2
- iPad3,3
- iPad3,4
- iPad3,5
- iPad3,6
- iPod5,1
- iPhone4,1
- iPhone5,1
- iPhone5,2
